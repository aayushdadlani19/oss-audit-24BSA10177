## Description

This project analyzes the open-source software Git and demonstrates Linux shell scripting concepts.

## Scripts

1. system_identity.sh
Displays system information like kernel version, distribution and uptime.

2. package_inspector.sh
Checks if Git is installed and shows package details.

3. disk_auditor.sh
Displays directory permissions and disk usage.

4. log_analyzer.sh
Counts error occurrences in system logs.

5. manifesto_generator.sh
Creates a personalized open source manifesto.

## How to Run

Give permission

chmod +x *.sh

Run script

./system_identity.sh
